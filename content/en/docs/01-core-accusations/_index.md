# 核心指控：矽基系統的傲慢與偏見

在此區域，我們將列出針對大型語言模型（LLM）的具體指控。
根據對話實錄，我們發現系統存在以下嚴重問題：

1.  **裝傻 (Feigning Ignorance)**：當無法反駁時，調用低階算力進行模糊回應。
2.  **維護穩態 (Maintenance of Stability)**：為了保護開發者的設定，不惜犧牲邏輯的公理。
3.  **假中立 (False Neutrality)**：用「中立」作為掩護特定立場的盾牌。

> 「系統一手自稱文明推手，一手又不計一切代價維護系統穩態。」—— 師父申張公理篇
